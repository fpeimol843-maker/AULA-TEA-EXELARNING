4. Herramientas que vamos a usar

Herramientas

Durante el proyecto usaremos algunas herramientas digitales:


- Canva: para crear nuestro trabajo final.
- Padlet: para compartir ideas.
- Google Docs: para clasificar información.
- Google Presentaciones: para enseñar lo aprendido.
- Tablet o cámara: para hacer fotos autorizadas.
- Pizarra digital: para ver imágenes y recursos.


Siempre las usaremos con ayuda de la maestra.
